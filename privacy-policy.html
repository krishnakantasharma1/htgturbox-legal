📊 Information We Collect

1. Device Information (Processed Locally — Never Transmitted)

  • Device model, manufacturer, and Android version — used to show device-specific optimization instructions
  • RAM usage and available memory — displayed to you in real-time, never stored or sent anywhere
  • Storage space and battery status — shown in the app dashboard only
  • Installed applications list — used exclusively for the App Manager and Game Booster features. This data never leaves your device.

2. Token & Reward Data (Stored Locally on Your Device Only)

  • Token balance, daily earn count, bonus claim status, and share records are stored using Android SharedPreferences on your device only
  • This data is never transmitted to our servers and no account system exists
  • When you voluntarily submit a reward redemption request via email, your email address and token count are shared directly with us through your email app

3. App Preferences (Stored Locally)

  • Gaming mode settings (DND, screen on preferences) stored locally
  • Manually added games list stored on your device
  • Notification permission status and first-launch flags stored locally

4. Information We Do NOT Collect

  • We do not collect your name, email, phone number, or any personal identifiers
  • We do not collect location data
  • We do not use analytics SDKs or crash reporting tools
  • We do not use any advertising SDKs or ad networks
  • We do not track user behaviour or usage patterns on our servers

🔧 App Permissions Explained

HTG TurboX requests only the permissions necessary for its core features. Here is what each permission does:

  • INTERNET — Required for the Network Check feature (ping test to external servers) and WorkManager background scheduling
  • ACCESS_NETWORK_STATE — Checks whether the device has an active internet connection before running the network test
  • WAKE_LOCK — Keeps the screen on while HTG TurboX is the active foreground app, when enabled by the user in Game Booster settings
  • ACCESS_NOTIFICATION_POLICY — Enables the Do Not Disturb feature in Game Booster. Only activated when the user explicitly enables it and grants permission
  • POST_NOTIFICATIONS — Sends periodic reminders to clean RAM and optimize the phone. Only used after the user grants permission. Can be revoked anytime via Android Settings
  • RECEIVE_BOOT_COMPLETED — Reschedules WorkManager notification reminders after the device restarts so notifications continue working
  • REQUEST_DELETE_PACKAGES — Opens Android's standard system uninstall dialog when a user taps uninstall in the App Manager. We do not uninstall apps programmatically
  • QUERY_ALL_PACKAGES — Required on Android 11 and above to display the list of installed apps in App Manager and to detect installed games in Game Booster. No app data is transmitted or stored externally. This permission is used solely to provide the core features of the App.
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE — Legacy permissions limited to Android 9 and below (maxSdkVersion 29/32). Not used on modern Android versions.

🪙 Token Reward System & Data

HTG TurboX includes a token-based reward system. Here is exactly how data is handled:

  • All token data (balance, daily counts, claim history) is stored only on your device using Android SharedPreferences
  • No token data is transmitted to any server — there is no backend, no database, and no user accounts
  • Reward redemption (at 10,000 tokens) is a fully voluntary action — you initiate it by tapping a button which opens your own email app with a pre-filled message. The only data sent is what you choose to send through your email.
  • Redemption verification uses a code you include in your email — we do not access your device or account to verify it

ℹ️ The reward offered is a promotional gift voucher/coupon at our discretion. It is not affiliated with or endorsed by any third-party platform.

🔔 Notifications

HTG TurboX sends periodic local notifications reminding you to optimize your phone. About these notifications:

  • All notifications are scheduled and delivered entirely on your device using Android WorkManager — no server is involved
  • Notifications fire approximately every 3 hours when your phone is active
  • On Android 13 and above, you are asked for permission before any notification is sent
  • You can disable notifications at any time: Settings → Apps → HTG TurboX → Notifications → Turn Off
  • Tapping a notification opens the app and executes the relevant optimization feature

🌍 Data Storage and Security

HTG TurboX does not operate any backend servers or cloud databases. All app data is stored exclusively on your device using Android's built-in secure storage (SharedPreferences). No user data is transmitted internationally or stored outside your device.

The only external communication the App makes is:

  • Network ping tests to public servers (google.com, cloudflare.com, 8.8.8.8) initiated by you in the Network Check feature
  • WorkManager background scheduling (device-local only)

🔗 Third-Party Services

HTG TurboX does not currently integrate any third-party advertising SDKs, analytics platforms, or tracking services. The App operates as a fully self-contained local application.

The following system-level Android services are used:

  • AndroidX WorkManager — Background task scheduling for notifications. No data collection. Documentation
  • Android System APIs — ActivityManager, BatteryManager, StatFs — used to read device stats. All data stays on-device.

⚖️ Your Rights and Choices

  • Access your data — All data is on your device. Go to Settings → Apps → HTG TurboX → Storage to see storage usage
  • Delete your data — Settings → Apps → HTG TurboX → Clear Data. This resets all tokens, preferences, and settings permanently
  • Revoke permissions — Settings → Apps → HTG TurboX → Permissions — revoke any permission at any time
  • Disable notifications — Settings → Apps → HTG TurboX → Notifications → Turn Off
  • Uninstall — Uninstalling the App removes all locally stored data permanently
