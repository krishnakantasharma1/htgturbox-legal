📊 Information We Collect

1. Device Information (Processed Locally — Never Transmitted)

The following device information is read by the App solely for the purpose of displaying statistics and optimization tips to you. This data is processed entirely on your device and is never stored persistently, transmitted to any server, or shared with any third party:

  • Device model, manufacturer, and Android version — used to show device-specific optimization instructions in the Tweaks section
  • RAM usage and available memory — displayed in real-time on the home screen dashboard
  • Storage capacity and free space — shown on the home screen dashboard
  • Battery level, charging status, and battery temperature — displayed in the dashboard and used by the Temperature Monitor feature
  • List of installed applications — used exclusively by the App Manager (to list user-installed apps for uninstallation) and the Game Booster (to detect installed games). This data never leaves your device.

2. Engagement Points Data (Stored Locally on Your Device Only)

  • Engagement point balance, daily earn count, and daily login bonus claim status are stored using Android SharedPreferences on your device only
  • This data is never transmitted to any server — there is no backend, no database, and no user accounts
  • Engagement points are purely cosmetic (badge milestones) and have no monetary value

3. App Preferences (Stored Locally)

  • Gaming mode settings (Do Not Disturb toggle, screen-on preference) — stored locally via SharedPreferences
  • Manually added games list — stored on your device via SharedPreferences
  • Notification permission status and first-launch flags — stored locally via SharedPreferences

4. Information We Do NOT Collect

  • We do not collect your name, email address, phone number, or any personal identifiers
  • We do not collect location data (GPS, IP-based, or otherwise)
  • We do not use analytics SDKs, crash reporting tools, or any server-side logging
  • We do not currently use any advertising SDKs or ad networks
  • We do not track user behavior, usage patterns, or session data on any server
  • We do not access, read, or transmit the contents of any other app on your device
  • We do not access your contacts, call logs, SMS, camera, microphone, or files

🔧 App Permissions Explained

HTG TurboX requests only the permissions necessary for its core features. Below is an explanation of each permission and why it is needed:

  • INTERNET — Required for the Network Check feature, which performs ping tests to public servers (google.com, cloudflare.com, 8.8.8.8) to measure your connection quality. No personal data is transmitted during these tests.
  • ACCESS_NETWORK_STATE — Checks whether the device has an active internet connection before running the Network Check feature. Does not collect or transmit any data.
  • ACCESS_NOTIFICATION_POLICY — Enables the Do Not Disturb (DND) feature in the Game Booster. DND is only activated when the user explicitly enables it via a toggle and grants the required system permission. The App never activates DND without user consent.
  • POST_NOTIFICATIONS (Android 13+) — Sends periodic informational reminders about checking device stats. Notifications are only sent after the user grants permission via Android's runtime permission dialog. Notifications can be disabled at any time via: Settings → Apps → HTG TurboX → Notifications → Turn Off.
  • RECEIVE_BOOT_COMPLETED — Reschedules the notification reminder service (via Android WorkManager) after the device restarts, so periodic reminders continue working. No data is collected or transmitted at boot.
  • QUERY_ALL_PACKAGES — Required on Android 11 and above to retrieve the list of installed applications. This permission is used solely for two features: (1) the App Manager, which displays user-installed apps and allows uninstallation via Android's standard system dialog, and (2) the Game Booster, which detects installed games to display them for quick launch. The list of installed apps is never stored persistently, transmitted to any server, or shared with any third party.

ℹ️ HTG TurboX does not request any permissions beyond those listed above. We follow the principle of minimum necessary permissions.

⭐ Engagement Points System & Data

HTG TurboX includes an engagement points system that allows users to earn points for using app features and unlock cosmetic badge milestones (Starter → Bronze → Silver → Gold → Diamond). Here is exactly how data is handled:

  • All engagement data (point balance, daily counts, bonus claim status) is stored only on your device using Android SharedPreferences
  • No engagement data is transmitted to any server — there is no backend, no database, and no user accounts
  • Engagement points have no monetary value and cannot be exchanged, transferred, or redeemed for real-world goods, services, or currency
  • Points and badges are purely cosmetic gamification features designed for fun
  • All engagement data is permanently deleted if you clear the App's data or uninstall the App

🔔 Notifications

HTG TurboX sends periodic local notifications with informational reminders to check your device stats. About these notifications:

  • All notifications are scheduled and delivered entirely on your device using Android WorkManager — no server is involved
  • Notifications are sent approximately every 8 hours when your phone is active
  • Notification content is informational only (e.g., "Tap to check your current RAM and storage usage") — notifications do not contain false claims, urgent warnings, or alarmist language
  • On Android 13 and above, explicit permission is requested before any notification is sent
  • You can disable notifications at any time: Settings → Apps → HTG TurboX → Notifications → Turn Off
  • Tapping a notification opens the App to the home screen
  • No personal data is used to target, personalize, or customize notification content

🌍 Data Storage and Security

HTG TurboX does not operate any backend servers, cloud databases, or remote storage. All app data is stored exclusively on your device using Android's built-in secure storage mechanisms (SharedPreferences within the App's private sandbox).

No user data is transmitted internationally, stored outside your device, or accessible to other apps.

The only external network communication the App makes is:

  • Network ping tests — Standard ICMP/TCP reachability checks to google.com, cloudflare.com, and 8.8.8.8, initiated only when you manually use the Network Check feature. No personal data is included in these requests.

🔗 Third-Party Services

HTG TurboX does not currently integrate any third-party advertising SDKs, analytics platforms, crash reporting services, or user tracking services. The App operates as a fully self-contained local application.

The following Android platform components are used:

  • AndroidX WorkManager — Android Jetpack library for scheduling periodic background tasks (notification reminders). Runs entirely on-device. No data collection. Documentation →
  • Android System APIs — ActivityManager (RAM stats), BatteryManager (battery info), StatFs (storage info), PackageManager (installed apps). All data stays on-device.

If third-party services (such as advertising SDKs) are integrated in future updates, this Privacy Policy will be updated accordingly before those changes take effect, and users will be notified through the App or via an updated version on the Google Play Store.

⚖️ Your Rights and Choices

Because HTG TurboX stores all data locally on your device and does not transmit any data to servers, you have full control over your data at all times:

  • Access your data — All data is on your device. Go to Settings → Apps → HTG TurboX → Storage to see storage usage.
  • Delete your data — Go to Settings → Apps → HTG TurboX → Clear Data. This permanently resets all engagement points, preferences, and settings.
  • Revoke permissions — Go to Settings → Apps → HTG TurboX → Permissions and revoke any permission at any time. The App will continue to function with reduced features.
  • Disable notifications — Go to Settings → Apps → HTG TurboX → Notifications → Turn Off.
  • Uninstall — Uninstalling the App removes all locally stored data permanently and completely.
