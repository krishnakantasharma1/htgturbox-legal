📱 Description of the App

HTG TurboX is a free Android gaming utility and device optimization application. The App provides:

Core Features (no special permissions required)

  • Home Dashboard — Real-time RAM, storage, battery, and temperature monitoring with health score
  • Device Specs — Detailed hardware information with copy-to-clipboard
  • Network Check — Ping tests, DNS resolution, connection type, and link speed
  • Performance Tweaks — Device-specific guides to Developer Options and system settings
  • Engagement Points — Cosmetic gamification with badge milestones

Game Booster Features

  • Game Launcher — Detect and quick-launch installed games
  • Gaming Mode — DND toggle, screen-on toggle, rotation lock toggle
  • Game Toolbox — Floating overlay with live stats (RAM, battery, temperature, FPS), quick toggles, and Boost button

Storage Analyzer

  • Storage Breakdown — View files by category (Photos, Videos, Audio, APK, Other) sorted by size
  • Duplicate Detection — Find duplicate files by matching exact byte size and MIME type
  • Multi-Select Deletion — Long-press to select multiple files for batch deletion via system confirmation
  • In-App Preview — View photos and videos without leaving the App
  • Largest Apps — View installed apps sorted by storage usage

App Manager

  • List user-installed apps and uninstall via Android's standard system dialog

Advanced Tweaks (Shizuku — Optional)

  • Animation Speed — Disable or reduce system animations
  • Background Process Limit — Limit background apps to free RAM
  • Max Cached Processes — Control how many recent apps stay in RAM
  • Performance Mode — Enable system performance mode on supported devices
  • Force GPU Rendering — Hardware-accelerate all apps
  • App Hibernation — Force-stop selected background apps
  • Screen Density (DPI) — Change display density
  • Disable Bloatware — Disable pre-installed apps without uninstalling
  • WiFi Scan Throttling — Stop constant background WiFi scanning
  • Disable HW Overlay — Force GPU composition
  • Refresh Rate Control — Force 60/90/120Hz display refresh rate
  • Clear All App Caches — Clear cached data for all apps safely

Module Flasher (Shizuku — Optional)

  • Load and execute .turbox module files containing preset configurations
  • Terminal-style interface showing command execution in real-time

ℹ️ HTG TurboX is a monitoring, optimization guidance, and system settings utility. Advanced tweaks modify Android's standard system settings database via ADB-level commands — the same settings accessible through Developer Options. The App does not modify system partitions, access root, or perform kernel-level operations.
